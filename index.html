Mercapp - Proyecto Full Stack

Resumen

Aplicación web con Vue 3, Node.js, Express y MongoDB Atlas. Desplegada en Railway (API) y Netlify (Frontend).

Arquitectura

  • Frontend: Vue 3 + Vite
  • Backend: Node.js + Express
  • DB: MongoDB Atlas

Enlaces del sistema

  • Frontend (Netlify): https://tu-netlify.app
  • API (Railway): https://tu-backend.up.railway.app

API Endpoints

Método Ruta Descripción
GET /api/products Lista productos
GET /api/categories Lista categorías
POST /api/products Crear producto
PUT /api/products/:id Actualizar producto
DELETE /api/products/:id Eliminar producto
